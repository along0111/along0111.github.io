PHP SDK

  • payment-php-sdk payment-php-sdk.zip

  • 第一步，在原有项目 composer.json中加入

    {
  "repositories": [
      {
          "type": "path", 
          "url": "<local_package_path>/asen/payment-php-sdk"
      }
  ]
}
  • 第二步执行以下shell安装
    composer require asen/payment-php-sdk:*@dev
  • 更多使用请关注pay_test.php、signature_test.php
